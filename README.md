# Sales Report Automation

This Python project automates the process of downloading a sales report, analyzing the data, and sending an email with a summary of the results.

⭐ Project developed for learning purposes.

## Technologies

- Python
- PyAutoGUI
- Pandas
- Pyperclip
- Jupyter Notebook

## Project Overview

This project was developed as practice for task automation with Python, combining UI automation, data manipulation, and automated email sending.

## How to Run

```bash
git clone https://github.com/LBaragao/sales_report_automation.git
cd sales_report_automation
python Inicial.py
```

## Install the dependencies

```bash
pip install -r requirements.txt
```

## Notes

- The click positions used by PyAutoGUI depend on screen resolution and browser layout, so they may need adjustments depending on the environment.
- Change `youremail@gmail.com` to your email address.
